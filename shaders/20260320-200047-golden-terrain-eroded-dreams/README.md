# Golden Terrain Eroded Dreams

A generative landscape that evolves over time, featuring an eroded terrain with a warm color palette of reds, oranges, and golds. The technique uses Perlin noise combined with erosion effects to create dynamic, flowing landscapes.

![Preview](https://raw.githubusercontent.com/ModusFactorum/glsl-art/main/shaders/20260320-200047-golden-terrain-eroded-dreams/preview.gif)

## Details

- **Rendered on:** NVIDIA RTX 5090 (headless EGL + moderngl)
- **Resolution:** 512x512
