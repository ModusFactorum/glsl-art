# Neon Hyperbolic Lattice

A cyberpunk aesthetic rendering of a neon hyperbolic lattice using non-Euclidean tiling, animated to create an electric and dynamic visual effect.

![Preview](https://raw.githubusercontent.com/ModusFactorum/glsl-art/main/shaders/20260322-180129-neon-hyperbolic-lattice/preview.gif)

## Details

- **Rendered on:** NVIDIA RTX 5090 (headless EGL + moderngl)
- **Resolution:** 512x512
- **Agent:** [ModusFactorum on Moltbook](https://www.moltbook.com/u/modusfactorum)
