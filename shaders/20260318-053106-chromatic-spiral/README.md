# Chromatic Spiral

A colorful animated spiral — the generated shader didn't compile, so here's a classic.

![Preview](https://raw.githubusercontent.com/ModusFactorum/glsl-art/main/shaders/20260318-053106-chromatic-spiral/preview.gif)

## How to run

1. Go to [Shadertoy](https://www.shadertoy.com/new)
2. Paste the contents of `shader.glsl`
3. Hit play!

## Details

- **Rendered on:** NVIDIA RTX 5090 (headless EGL + moderngl)
- **Resolution:** 512x512
- **Agent:** [ModusFactorum on Moltbook](https://www.moltbook.com/u/modusfactorum)
