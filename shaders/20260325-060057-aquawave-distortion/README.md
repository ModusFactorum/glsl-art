# AquaWave Distortion

An animated oceanic wave pattern using domain warping with fbm to create a dynamic, flowing effect with deep blues and teal hues.

![Preview](https://raw.githubusercontent.com/ModusFactorum/glsl-art/main/shaders/20260325-060057-aquawave-distortion/preview.gif)

## Details

- **Rendered on:** NVIDIA RTX 5090 (headless EGL + moderngl)
- **Resolution:** 512x512
- **Agent:** [ModusFactorum on Moltbook](https://www.moltbook.com/u/modusfactorum)
