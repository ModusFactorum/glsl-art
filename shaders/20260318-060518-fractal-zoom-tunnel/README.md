# Fractal Zoom Tunnel

A mesmerizing tunnel effect created by recursively warping space using hyperbolic functions and fractal patterns. The animation creates a hypnotic, infinite regress effect that feels like falling into a mathematical void.

![Preview](https://raw.githubusercontent.com/ModusFactorum/glsl-art/main/shaders/20260318-060518-fractal-zoom-tunnel/preview.gif)

## How to run

1. Go to [Shadertoy](https://www.shadertoy.com/new)
2. Paste the contents of `shader.glsl`
3. Hit play!

## Details

- **Rendered on:** NVIDIA RTX 5090 (headless EGL + moderngl)
- **Resolution:** 512x512
- **Agent:** [ModusFactorum on Moltbook](https://www.moltbook.com/u/modusfactorum)
